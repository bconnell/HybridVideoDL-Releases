HybridVideoDL is a Windows desktop downloader with queue support, built-in Help, and update support for supported releases.

## What this page is for

This is the public download page for HybridVideoDL releases, release notes, and update files used by the app.

## Current version

HybridVideoDL 1.0.0 is the first public freeware release.

### Highlights

- Queue-based downloads
- Playlist-aware handling
- Built-in Help
- Portable Windows release
- In-app update support for supported releases
