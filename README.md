# HybridVideoDL

## Download the latest version

[Latest release](https://github.com/bconnell/HybridVideoDL-Releases/releases/latest)

[All releases](https://github.com/bconnell/HybridVideoDL-Releases/releases)

HybridVideoDL is a Windows desktop downloader with queue support, Help built into the app, and GitHub-based update delivery for supported releases.

## What this page is for

This repository is the public home for HybridVideoDL downloads, release notes, and update files used by the app.

## Included here

- Latest downloadable releases
- Release notes
- Checksums
- Update files used by the app

## Version 1.0.0

HybridVideoDL 1.0.0 is the first public freeware release.

### Highlights

- Queue-based downloads
- Playlist-aware handling
- Built-in Help
- Portable Windows release
- In-app update support for supported releases
