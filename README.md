# HybridVideoDL Releases

HybridVideoDL is a Windows desktop downloader built around `yt-dlp`, with queue support, playlist handling, built-in Help, and update support for official releases.

## Download

[Latest release](https://github.com/bconnell/HybridVideoDL-Releases/releases/latest)

[All releases](https://github.com/bconnell/HybridVideoDL-Releases/releases)

## Use the portable build

1. Download the latest portable zip
2. Extract it to a folder you control
3. Keep the extracted files together
4. Run `HybridVideoDL.exe`
