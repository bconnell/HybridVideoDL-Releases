# HybridVideoDL Releases

This public repository is for HybridVideoDL distribution files only.

## What belongs here

- Portable zip packages
- Update manifests
- Checksums
- User-facing release notes

## What does not belong here

- Source code
- Solution or project files
- Tests
- Developer notes
- Local machine files
- Secrets or credentials

## Current release feed

- Update manifest: updates/stable.json
- Portable packages: artifacts/
- Release notes: release-notes/
- Checksums: checksums/

Repository URL: https://github.com/bconnell/HybridVideoDL-Releases
