# HybridVideoDL Releases

Official public downloads for HybridVideoDL, a Windows desktop downloader built around yt-dlp.

## Download

Official end-user downloads are published in the GitHub Releases section for this repository.

- [Latest release](https://github.com/bconnell/HybridVideoDL-Releases/releases/latest)
- [HybridVideoDL 1.0.1 portable zip](https://github.com/bconnell/HybridVideoDL-Releases/releases/download/v1.0.1/HybridVideoDL_v1.0.1_portable_win-x64.zip)

Do not browse repository folders for normal downloads. Release assets are attached to each GitHub Release.

## Repository structure

- `Releases` = downloadable release assets for end users
- `updates/` = updater metadata only
- `release-notes/` = release notes only
- `checksums/` = checksum metadata only

## Using the portable build

1. Download the portable zip from GitHub Releases
2. Extract the zip to a folder you control
3. Keep the extracted files together
4. Run `HybridVideoDL.exe`
5. Open `Settings` and choose your local `yt-dlp.exe` path before downloading

Some features require user-supplied third-party tools. Downloads need `yt-dlp`, and conversion features need `ffmpeg`. The app links directly to the official download pages from `Settings`.
