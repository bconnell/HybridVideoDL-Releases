# HybridVideoDL Releases

HybridVideoDL Releases is the official public download repository for HybridVideoDL.

HybridVideoDL is a freeware Windows desktop downloader with queue support, built-in Help, and update support for supported releases.

## Download HybridVideoDL

[Latest release](https://github.com/bconnell/HybridVideoDL-Releases/releases/latest)

[All releases](https://github.com/bconnell/HybridVideoDL-Releases/releases)

- Portable packages are published in `artifacts/`
- User-facing release notes are published in `release-notes/`
- The built-in updater reads `updates/stable.json`

## Using the portable build

1. Download the latest portable zip
2. Extract it to a folder you control
3. Keep the extracted files together
4. Run `HybridVideoDL.exe`

## Project and source code

- Main project repository: https://github.com/bconnell/HybridVideoDL
- Source code development happens in the main project repository
- This release repository is distribution-only and is not a mirror of the source repo
