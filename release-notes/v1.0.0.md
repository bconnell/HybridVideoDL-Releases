# HybridVideoDL 1.0.0

HybridVideoDL 1.0.0 is the first public freeware release of the Windows desktop downloader.

## Highlights

- Queue-based downloads powered by yt-dlp
- Playlist-aware output handling
- Portable folder distribution with bundled runtime files
- In-app Help, donation popout, and queue cleanup controls
- GitHub-based update checks for the supported portable package

## Update notes

- Updates preserve app settings and queue data by design
- Portable mode keeps working data under AppData beside the executable
- Standard mode keeps working data under %LocalAppData%\HybridVideoDL
