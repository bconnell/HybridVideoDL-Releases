# HybridVideoDL 1.0.1

HybridVideoDL 1.0.1 is the corrected public portable release for the Windows desktop downloader.

## Highlights

- Queue-based downloads powered by yt-dlp
- Playlist-aware output handling
- Portable folder distribution with the Windows desktop runtime included
- In-app Help, donation popup, and queue cleanup controls
- GitHub-based update checks for the supported portable package

## Tool setup

- HybridVideoDL does not include `yt-dlp`, `ffmpeg`, or `ffprobe`
- Downloads require a user-supplied `yt-dlp.exe` configured in `Settings`
- Conversion features require a user-supplied `ffmpeg.exe` configured in `Settings`
- Settings includes direct links to the official `yt-dlp` and `FFmpeg` download pages

## Release notes

- Updates preserve app settings and queue data by design
- Portable mode keeps working data under `AppData` beside the executable
- Standard mode keeps working data under `%LocalAppData%\HybridVideoDL`
- This package is the supported public portable release
